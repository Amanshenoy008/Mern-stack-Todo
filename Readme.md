### Todo App 

It is even mobile responsive

Frontend:
ReactJs , Tailwindcss + DaisyUI + Reactmui 

Backend:
NodeJs , ExpressJs , Mongoose ORM , MongoDB 


In order to run 

open backend folder

```
    npm i 
    npm start
```
open frontend folder

```
    npm i
    nodemon server.js
```

The Site will be live on http://localhost:3000
